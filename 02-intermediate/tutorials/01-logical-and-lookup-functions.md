# 逻辑与查找函数

## 一、逻辑函数进阶

### 1.1 IF嵌套

当需要多个条件判断时，可以嵌套使用IF函数。

**语法：**
```
=IF(条件1, 结果1, IF(条件2, 结果2, IF(条件3, 结果3, 默认结果)))
```

**示例：**
```
=IF(A1>=90,"优秀",IF(A1>=80,"良好",IF(A1>=60,"及格","不及格")))
```

**注意：** Excel 2003及更早版本最多支持7层嵌套，Excel 2007+支持64层。

### 1.2 IFS函数（Excel 2019+）

IFS可以替代多层IF嵌套，更加简洁。

**语法：**
```
=IFS(条件1, 结果1, 条件2, 结果2, ..., 默认结果)
```

**示例：**
```
=IFS(A1>=90,"优秀",A1>=80,"良好",A1>=60,"及格",TRUE,"不及格")
```

**对比IF嵌套：**
| 函数 | 特点 |
|------|------|
| IF嵌套 | 兼容性好，写法较繁琐 |
| IFS | 写法简洁，需要Excel 2019+ |

### 1.3 AND/OR函数

**AND函数**：所有条件都为真时返回TRUE

**语法：**
```
=AND(条件1, 条件2, ...)
```

**示例：**
```
=AND(A1>=60, B1>=60, C1>=60)  // 三门课都及格
```

**OR函数**：任一条件为真时返回TRUE

**语法：**
```
=OR(条件1, 条件2, ...)
```

**示例：**
```
=OR(A1>=90, B1>=90, C1>=90)  // 至少一门课优秀
```

### 1.4 NOT函数

NOT函数：对条件取反

**语法：**
```
=NOT(条件)
```

**示例：**
```
=NOT(A1="")  // A1不为空时返回TRUE
```

## 二、查找函数

### 2.1 VLOOKUP

VLOOKUP是最常用的查找函数，按列查找。

**语法：**
```
=VLOOKUP(查找值, 查找区域, 返回列数, [匹配类型])
```

**参数说明：**
- **查找值**：要查找的值
- **查找区域**：包含查找值和返回值的区域，查找值必须在第一列
- **返回列数**：从查找区域第一列开始数，要返回的列序号
- **匹配类型**：FALSE（精确匹配）或TRUE（近似匹配，默认）

**示例：**
```
=VLOOKUP(A2, 员工表!A:B, 2, FALSE)  // 在员工表中查找A2的姓名
```

**VLOOKUP的局限性：**
1. 只能从左向右查找
2. 查找值必须在第一列
3. 列顺序变化会导致错误

### 2.2 XLOOKUP（Excel 2021+）

XLOOKUP是VLOOKUP的升级版，功能更强大。

**语法：**
```
=XLOOKUP(查找值, 查找列, 返回列, [未找到值], [匹配模式], [搜索模式])
```

**参数说明：**
- **查找值**：要查找的值
- **查找列**：包含查找值的列（单列）
- **返回列**：要返回的值所在的列（单列）
- **未找到值**：可选，查找不到时返回的值
- **匹配模式**：0=精确匹配（默认），-1=精确匹配或下一个更小，1=精确匹配或下一个更大，2=通配符匹配
- **搜索模式**：1=从上到下（默认），-1=从下到上，2=二分查找（升序），-2=二分查找（降序）

**示例：**
```
=XLOOKUP(A2, 员工表!A:A, 员工表!B:B, "未找到", 0)  // 精确查找姓名
```

**XLOOKUP优势：**
1. 可以从右向左查找
2. 查找列和返回列可以是任意列
3. 提供未找到值的默认返回
4. 支持通配符匹配

### 2.3 INDEX+MATCH组合

INDEX+MATCH是功能最强大的查找组合，不受列顺序限制。

**INDEX函数**：根据行号和列号返回单元格值

**语法：**
```
=INDEX(返回区域, 行号, [列号])
```

**MATCH函数**：返回查找值在区域中的位置

**语法：**
```
=MATCH(查找值, 查找区域, [匹配类型])
```

**组合使用：**
```
=INDEX(员工表!B:B, MATCH(A2, 员工表!A:A, 0))  // 查找姓名
=INDEX(员工表!C:C, MATCH(A2, 员工表!A:A, 0))  // 查找部门
```

**多条件查找：**
```
=INDEX(销售表!D:D, MATCH(1, (A2=销售表!A:A)*(B2=销售表!B:B), 0))
```

**INDEX+MATCH优势：**
1. 可以从任意方向查找
2. 不受列顺序限制
3. 支持多条件查找
4. 兼容性好（所有Excel版本）

## 三、查找函数对比

| 函数 | 方向限制 | 版本要求 | 多条件 | 推荐度 |
|------|----------|----------|--------|--------|
| VLOOKUP | 只能从左向右 | 所有版本 | 不支持 | ⭐⭐⭐ |
| XLOOKUP | 无限制 | Excel 2021+ | 支持 | ⭐⭐⭐⭐⭐ |
| INDEX+MATCH | 无限制 | 所有版本 | 支持 | ⭐⭐⭐⭐⭐ |

## 四、示例练习

### 4.1 成绩等级判定

**场景：** 根据分数判定等级

**公式：**
```
=IFS(A1>=90,"优秀",A1>=80,"良好",A1>=60,"及格",TRUE,"不及格")
```

### 4.2 员工信息查询

**场景：** 根据员工ID查询姓名和部门

**数据：**
| ID | 姓名 | 部门 |
|----|------|------|
| 001 | 张三 | 销售部 |
| 002 | 李四 | 技术部 |

**公式（VLOOKUP）：**
```
=VLOOKUP(A2, 员工表!A:C, 2, FALSE)  // 查询姓名
=VLOOKUP(A2, 员工表!A:C, 3, FALSE)  // 查询部门
```

**公式（XLOOKUP）：**
```
=XLOOKUP(A2, 员工表!A:A, 员工表!B:B, "未找到", 0)
```

**公式（INDEX+MATCH）：**
```
=INDEX(员工表!B:B, MATCH(A2, 员工表!A:A, 0))
```

### 4.3 多条件查找

**场景：** 根据产品和区域查找销售额

**数据：**
| 产品 | 区域 | 销售额 |
|------|------|--------|
| 产品A | 华北 | 10000 |
| 产品A | 华东 | 12000 |

**公式：**
```
=INDEX(C:C, MATCH(1, (A2=A:A)*(B2=B:B), 0))
```

## 五、常见错误与技巧

### 5.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `#N/A` | 未找到匹配值 | 检查查找值是否正确，使用IFERROR处理 |
| `#REF!` | 引用列超出范围 | 检查返回列数是否正确 |
| 结果错误 | 匹配类型错误 | 使用FALSE进行精确匹配 |

### 5.2 实用技巧

1. **处理#N/A错误**：`=IFERROR(VLOOKUP(...), "未找到")`
2. **模糊查找**：使用通配符 `*` 和 `?`
3. **反向查找**：使用INDEX+MATCH或XLOOKUP
4. **近似匹配**：用于区间查找（如税率表）

---

**适用版本**：Excel 2016+, Microsoft 365（XLOOKUP/IFS需要Excel 2021+）
