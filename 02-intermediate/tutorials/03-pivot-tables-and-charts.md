# 数据透视表与图表

## 一、数据透视表基础

### 1.1 创建数据透视表

**操作步骤：**
1. 选中数据区域（包含标题行）
2. 「插入」选项卡 → 「数据透视表」
3. 在弹出的对话框中确认数据区域
4. 选择放置位置（新建工作表或现有工作表）
5. 点击「确定」

### 1.2 字段布局

数据透视表包含四个区域：

| 区域 | 说明 | 常用字段 |
|------|------|----------|
| **筛选器（Filters）** | 全局筛选 | 日期、区域 |
| **行（Rows）** | 行标签 | 产品、部门 |
| **列（Columns）** | 列标签 | 月份、类别 |
| **值（Values）** | 计算值 | 销售额、数量 |

**操作方式：**
- 拖拽字段到相应区域
- 双击字段可以设置字段属性
- 右键字段可以删除或移动

### 1.3 值汇总方式

**默认汇总方式：**
- 数值字段：求和（SUM）
- 文本字段：计数（COUNT）

**更改汇总方式：**
1. 右键值区域的任意单元格
2. 选择「值字段设置」
3. 在弹出的对话框中选择汇总方式

**常用汇总方式：**

| 汇总方式 | 说明 |
|----------|------|
| 求和 | 计算总和 |
| 计数 | 计算个数 |
| 平均值 | 计算平均值 |
| 最大值 | 返回最大值 |
| 最小值 | 返回最小值 |
| 乘积 | 计算乘积 |
| 标准差 | 计算标准差 |

## 二、数据透视表高级功能

### 2.1 筛选与切片器

**筛选器（Filters）：**
1. 将字段拖到「筛选器」区域
2. 点击筛选器下拉箭头选择筛选条件

**切片器（Slicers）：**
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「插入切片器」
3. 选择要创建切片器的字段
4. 点击「确定」

**切片器优点：**
- 可视化筛选界面
- 支持多选
- 可以连接多个数据透视表

### 2.2 分组功能

**日期分组：**
1. 右键日期字段
2. 选择「分组」
3. 在弹出的对话框中选择分组方式（年、季度、月、日）
4. 点击「确定」

**数值分组：**
1. 右键数值字段
2. 选择「分组」
3. 设置起始值、终止值和步长
4. 点击「确定」

### 2.3 计算字段

**创建计算字段：**
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「字段、项目和集」→「计算字段」
3. 输入字段名称
4. 在公式框中输入计算表达式
5. 点击「添加」→「确定」

**示例：**
```
利润率 = 利润 / 销售额
平均单价 = 销售额 / 数量
```

### 2.4 数据透视图

**创建数据透视图：**
1. 选中数据透视表
2. 「分析」选项卡 → 「数据透视图」
3. 选择图表类型
4. 点击「确定」

**数据透视图特点：**
- 与数据透视表联动
- 支持切片器筛选
- 图表会随透视表更新

## 三、图表基础

### 3.1 选择合适的图表类型

| 图表类型 | 适用场景 |
|----------|----------|
| **柱形图/条形图** | 比较不同类别的数值 |
| **折线图** | 展示数据随时间变化趋势 |
| **饼图/环形图** | 展示各部分占总体的比例 |
| **散点图** | 展示两个变量之间的关系 |
| **面积图** | 展示数据随时间的累积变化 |
| **雷达图** | 比较多个维度的数据 |

### 3.2 创建图表

**操作步骤：**
1. 选中数据区域（包含标题行）
2. 「插入」选项卡 → 选择图表类型
3. Excel自动创建图表
4. 在「图表设计」和「格式」选项卡中调整样式

### 3.3 图表元素

**图表元素包括：**
- 标题（Chart Title）
- 坐标轴（Axis）
- 图例（Legend）
- 数据标签（Data Labels）
- 网格线（Gridlines）
- 数据表（Data Table）

**添加图表元素：**
1. 选中图表
2. 点击图表右侧的「+」按钮
3. 勾选要添加的元素

### 3.4 图表美化

**常用美化操作：**
1. 更改图表样式：「图表设计」→「图表样式」
2. 更改颜色：「图表设计」→「颜色」
3. 添加数据标签：勾选「数据标签」
4. 设置坐标轴格式：右键坐标轴 →「设置坐标轴格式」
5. 添加趋势线：「图表设计」→「添加图表元素」→「趋势线」

## 四、示例练习

### 4.1 创建销售数据透视表

**场景：** 分析各区域各产品的销售情况

**数据：** 销售数据表（产品、区域、销售额、日期）

**操作步骤：**
1. 选中数据区域
2. 「插入」→「数据透视表」
3. 将「区域」拖到行区域
4. 将「产品」拖到列区域
5. 将「销售额」拖到值区域
6. 添加切片器筛选日期

### 4.2 创建销售趋势图

**场景：** 展示月度销售趋势

**操作步骤：**
1. 选中日期和销售额列
2. 「插入」→「折线图」
3. 添加图表标题
4. 添加数据标签
5. 设置日期格式

### 4.3 创建销售占比饼图

**场景：** 展示各产品销售额占比

**操作步骤：**
1. 选中产品和销售额列
2. 「插入」→「饼图」
3. 添加数据标签（显示百分比）
4. 设置图例位置

## 五、常见错误与技巧

### 5.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 数据透视表不更新 | 源数据变化后未刷新 | 右键透视表 →「刷新」 |
| 日期字段不分组 | 日期格式不正确 | 检查并设置为日期格式 |
| 图表数据错误 | 数据源选择错误 | 重新选择数据区域 |
| 切片器不显示 | 未启用切片器功能 | 「插入切片器」重新创建 |

### 5.2 实用技巧

1. **刷新所有数据透视表**：`Ctrl+Alt+F5`
2. **移动数据透视表**：右键 →「移动数据透视表」
3. **复制数据透视表**：复制粘贴到新位置
4. **清除数据透视表**：右键 →「清除」→「全部清除」
5. **创建迷你图**：选中单元格 →「插入」→「迷你图」
