# 文本、日期与数组函数

## 一、文本函数

### 1.1 LEFT/RIGHT/MID

**LEFT函数**：从文本左侧提取指定长度的字符

**语法：**
```
=LEFT(文本, [提取长度])
```

**示例：**
```
=LEFT("Hello World", 5)  // 返回 "Hello"
=LEFT(A1)                // 返回第一个字符
```

**RIGHT函数**：从文本右侧提取指定长度的字符

**语法：**
```
=RIGHT(文本, [提取长度])
```

**示例：**
```
=RIGHT("Hello World", 5)  // 返回 "World"
=RIGHT(A1, 4)             // 返回最后4个字符
```

**MID函数**：从文本中间提取指定长度的字符

**语法：**
```
=MID(文本, 起始位置, 提取长度)
```

**示例：**
```
=MID("Hello World", 7, 5)  // 返回 "World"
=MID(A1, 2, 3)             // 从第2位开始提取3个字符
```

### 1.2 LEN/LENB

**LEN函数**：返回文本长度（字符数）

**语法：**
```
=LEN(文本)
```

**示例：**
```
=LEN("Hello")  // 返回 5
=LEN(A1)       // 返回A1的字符长度
```

**LENB函数**：返回文本长度（字节数，中文占2字节）

**语法：**
```
=LENB(文本)
```

**示例：**
```
=LENB("你好")   // 返回 4（2个中文 × 2字节）
=LENB("Hello")  // 返回 5（5个英文 × 1字节）
```

### 1.3 TEXT函数

TEXT函数：将数字或日期转换为指定格式的文本

**语法：**
```
=TEXT(值, 格式代码)
```

**日期格式代码：**

| 格式代码 | 说明 | 示例 |
|----------|------|------|
| yyyy | 四位数年份 | 2024 |
| mm | 两位数月份 | 01-12 |
| dd | 两位数日期 | 01-31 |
| dddd | 星期几 | 星期一 |
| hh | 两位数小时 | 00-23 |
| mm | 两位数分钟 | 00-59 |
| ss | 两位数秒 | 00-59 |

**示例：**
```
=TEXT(TODAY(), "yyyy年mm月dd日")  // 返回 "2024年01月15日"
=TEXT(A1, "¥#,##0")              // 返回 "¥12,345"
=TEXT(NOW(), "hh:mm:ss")         // 返回当前时间
```

### 1.4 FIND/SEARCH

**FIND函数**：查找文本在另一文本中的位置（区分大小写）

**语法：**
```
=FIND(要查找的文本, 包含文本, [起始位置])
```

**示例：**
```
=FIND("o", "Hello")       // 返回 5
=FIND("O", "Hello")       // 返回错误（区分大小写）
=FIND("l", "Hello", 3)    // 从第3位开始查找，返回 4
```

**SEARCH函数**：查找文本在另一文本中的位置（不区分大小写）

**语法：**
```
=SEARCH(要查找的文本, 包含文本, [起始位置])
```

**示例：**
```
=SEARCH("O", "Hello")     // 返回 5（不区分大小写）
```

## 二、日期函数

### 2.1 TODAY/NOW

**TODAY函数**：返回当前日期

**语法：**
```
=TODAY()
```

**NOW函数**：返回当前日期和时间

**语法：**
```
=NOW()
```

**示例：**
```
=TODAY()          // 返回 2024/01/15
=NOW()            // 返回 2024/01/15 14:30:00
=TODAY()+7        // 返回7天后的日期
=TODAY()-A1       // 计算日期差
```

### 2.2 YEAR/MONTH/DAY

**YEAR函数**：提取年份

**语法：**
```
=YEAR(日期)
```

**示例：**
```
=YEAR(TODAY())   // 返回 2024
=YEAR(A1)        // 返回A1日期的年份
```

**MONTH函数**：提取月份

**语法：**
```
=MONTH(日期)
```

**示例：**
```
=MONTH(TODAY())  // 返回 1（1月）
```

**DAY函数**：提取日期

**语法：**
```
=DAY(日期)
```

**示例：**
```
=DAY(TODAY())    // 返回 15
```

### 2.3 EOMONTH

EOMONTH函数：返回指定月份的最后一天

**语法：**
```
=EOMONTH(开始日期, 月份数)
```

**示例：**
```
=EOMONTH(TODAY(), 0)    // 返回本月最后一天
=EOMONTH(TODAY(), 1)    // 返回下月最后一天
=EOMONTH(TODAY(), -1)   // 返回上月最后一天
```

### 2.4 NETWORKDAYS

NETWORKDAYS函数：计算两个日期之间的工作日数（排除周末）

**语法：**
```
=NETWORKDAYS(开始日期, 结束日期, [节假日])
```

**示例：**
```
=NETWORKDAYS(A1, B1)        // 计算A1到B1的工作日数
=NETWORKDAYS(A1, B1, C1:C10) // 排除指定节假日
```

### 2.5 DATEDIF

DATEDIF函数：计算两个日期之间的差值

**语法：**
```
=DATEDIF(开始日期, 结束日期, 单位)
```

**单位参数：**

| 单位 | 说明 |
|------|------|
| "Y" | 年数 |
| "M" | 月数 |
| "D" | 天数 |
| "MD" | 天数（忽略年月） |
| "YM" | 月数（忽略年日） |
| "YD" | 天数（忽略年） |

**示例：**
```
=DATEDIF(A1, TODAY(), "Y")   // 计算年龄（年）
=DATEDIF(A1, TODAY(), "M")   // 计算月份差
=DATEDIF(A1, TODAY(), "D")   // 计算天数差
```

## 三、动态数组函数（Excel 2021+）

### 3.1 FILTER

FILTER函数：根据条件筛选数据

**语法：**
```
=FILTER(数组, 条件, [未找到值])
```

**示例：**
```
=FILTER(A2:C10, B2:B10="销售部")    // 筛选销售部的数据
=FILTER(A2:C10, C2:C10>10000)      // 筛选销售额>10000的数据
=FILTER(A2:C10, B2:B10="销售部", "无数据") // 无匹配时显示"无数据"
```

### 3.2 SORT

SORT函数：对数据进行排序

**语法：**
```
=SORT(数组, [排序列], [排序顺序], [按列排序])
```

**示例：**
```
=SORT(A2:C10)                     // 按第一列升序排序
=SORT(A2:C10, 3, -1)              // 按第三列降序排序（-1=降序）
=SORT(A2:C10, {1,3}, {1,-1})      // 先按第一列升序，再按第三列降序
```

### 3.3 UNIQUE

UNIQUE函数：提取唯一值

**语法：**
```
=UNIQUE(数组, [仅返回唯一值], [按列比较])
```

**示例：**
```
=UNIQUE(A2:A10)                   // 提取A列的唯一值
=UNIQUE(A2:C10)                   // 提取唯一的行
=UNIQUE(A2:A10, TRUE)             // 仅返回出现一次的值
```

### 3.4 SEQUENCE

SEQUENCE函数：生成数字序列

**语法：**
```
=SEQUENCE(行数, [列数], [起始值], [步长])
```

**示例：**
```
=SEQUENCE(10)                     // 生成1-10
=SEQUENCE(5, 3)                   // 生成5行3列的序列
=SEQUENCE(10, 1, 100, 10)        // 生成100,110,...,190
```

### 3.5 XLOOKUP多条件

**语法：**
```
=XLOOKUP(1, (条件1)*(条件2), 返回列)
```

**示例：**
```
=XLOOKUP(1, (A2:A10="产品A")*(B2:B10="华北"), C2:C10)
```

## 四、示例练习

### 4.1 从身份证号提取信息

**场景：** 从身份证号提取出生日期和性别

**数据：** A1 = "110101199003075678"

**公式：**
```
=TEXT(MID(A1, 7, 8), "yyyy/mm/dd")  // 提取出生日期
=IF(MOD(MID(A1, 17, 1), 2)=1, "男", "女")  // 判断性别
```

### 4.2 计算员工工龄

**场景：** 根据入职日期计算工龄

**公式：**
```
=DATEDIF(B2, TODAY(), "Y") & "年" & DATEDIF(B2, TODAY(), "YM") & "个月"
```

### 4.3 筛选数据

**场景：** 筛选销售部员工

**公式：**
```
=FILTER(A2:D10, C2:C10="销售部")
```

### 4.4 排序数据

**场景：** 按薪资降序排序

**公式：**
```
=SORT(A2:D10, 4, -1)
```

## 五、常见错误与技巧

### 5.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `#CALC!` | 动态数组溢出 | 检查返回区域是否有足够空间 |
| `#VALUE!` | 参数类型错误 | 检查参数是否为正确的文本或日期 |
| 日期显示为数字 | 格式问题 | 设置单元格格式为日期 |

### 5.2 实用技巧

1. **提取年份后两位**：`=RIGHT(YEAR(A1), 2)`
2. **获取本周一日期**：`=TODAY()-WEEKDAY(TODAY(), 2)+1`
3. **获取本月第一天**：`=EOMONTH(TODAY(), -1)+1`
4. **动态数组结果清除**：选中结果区域 → Delete键
