# 进阶阶段 - 函数、数据透视表与图表

## 阶段目标

本阶段旨在帮助学习者掌握 Excel 中更高级的函数和数据分析工具，包括逻辑函数、查找函数、文本日期函数、数据透视表和图表等核心技能。

## 学习内容

1. **逻辑与查找函数** - IF嵌套、IFS、AND/OR、VLOOKUP、XLOOKUP、INDEX+MATCH
2. **文本、日期与数组函数** - LEFT/RIGHT/MID、TEXT、TODAY/EOMONTH、NETWORKDAYS、动态数组
3. **数据透视表与图表** - 创建透视表、字段布局、筛选与切片器、数据透视图
4. **数据验证与假设分析** - 下拉列表、输入限制、模拟运算、方案管理器

## 适用版本

- Excel 2016+（部分功能需要 Excel 2021+ 或 Microsoft 365）
- Microsoft 365（推荐）

## 学习建议

1. 先掌握逻辑和查找函数，这是数据分析的基础
2. 数据透视表是高效分析数据的利器，务必多练习
3. 图表能直观展示数据，学习选择合适的图表类型
4. 完成练习后对照答案检查，重点理解操作思路

## 版本注意事项

- `XLOOKUP`、`IFS`、`FILTER`、`SORT`、`UNIQUE` 等函数需要 Excel 2021+ 或 Microsoft 365
- 如果使用旧版 Excel，可以用 `VLOOKUP`、嵌套 `IF` 等替代
