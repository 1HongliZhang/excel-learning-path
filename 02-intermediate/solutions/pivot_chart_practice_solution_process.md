# pivot_chart_practice.xlsx 练习解决方案

## 练习目标

1. 创建数据透视表分析多维度数据
2. 创建数据透视图
3. 添加切片器进行交互式筛选
4. 使用图表展示数据关系

## 数据说明

- **列A（月份）**：1月-12月
- **列B（产品类别）**：电子产品、服装鞋帽、食品饮料、家居用品、办公用品
- **列C（地区）**：北京、上海、广州、深圳、成都、杭州
- **列D（销售额）**：销售金额
- **列E（利润）**：利润金额

## 解决方案步骤

### Step 1: 创建基础数据透视表

**目标**：分析各地区各产品类别的销售情况

**操作**：
1. 选中数据区域（A1:E361）
2. 「插入」选项卡 → 「数据透视表」
3. 选择放置位置：新建工作表
4. 点击「确定」

### Step 2: 配置字段布局

**目标**：设置行、列、值字段

**操作**：
1. 将「地区」字段拖到「行」区域
2. 将「产品类别」字段拖到「列」区域
3. 将「销售额」字段拖到「值」区域
4. 将「利润」字段拖到「值」区域

**结果说明**：
- 行显示各地区
- 列显示各产品类别和总计
- 值显示销售额和利润的总和

### Step 3: 添加月份切片器

**目标**：添加月份切片器进行时间筛选

**操作**：
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「插入切片器」
3. 勾选「月份」字段
4. 点击「确定」

**自定义切片器样式**：
1. 选中切片器
2. 在「切片器工具」选项卡中选择合适的样式
3. 调整切片器大小和位置

### Step 4: 创建数据透视图

**目标**：创建柱状图展示各地区销售对比

**操作**：
1. 选中数据透视表
2. 「分析」选项卡 → 「数据透视图」
3. 在弹出的对话框中选择「柱形图」→「簇状柱形图」
4. 点击「确定」

**调整数据透视图**：
1. 添加图表标题：「各地区各产品类别销售额对比」
2. 添加数据标签
3. 调整图例位置
4. 设置坐标轴格式

### Step 5: 创建利润分析数据透视表

**目标**：分析各产品类别的利润率

**操作**：
1. 再次创建一个数据透视表
2. 将「产品类别」字段拖到「行」区域
3. 将「销售额」字段拖到「值」区域
4. 将「利润」字段拖到「值」区域

**创建计算字段**：
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「字段、项目和集」→「计算字段」
3. 名称：利润率
4. 公式：`=利润 / 销售额`
5. 点击「添加」→「确定」

**设置百分比格式**：
1. 右键「利润率」列的任意单元格
2. 选择「值字段设置」
3. 点击「数字格式」
4. 设置为百分比格式，保留2位小数

### Step 6: 创建利润趋势数据透视图

**目标**：创建折线图展示月度利润变化

**操作**：
1. 创建新的数据透视表
2. 将「月份」字段拖到「行」区域
3. 将「利润」字段拖到「值」区域
4. 选中数据透视表
5. 「分析」选项卡 → 「数据透视图」
6. 选择「折线图」→「带数据标记的折线图」
7. 添加图表标题：「月度利润趋势图」
8. 添加数据标签

### Step 7: 创建销售占比环形图

**目标**：展示各地区销售占比

**操作**：
1. 创建新的数据透视表
2. 将「地区」字段拖到「行」区域
3. 将「销售额」字段拖到「值」区域
4. 选中数据透视表
5. 「分析」选项卡 → 「数据透视图」
6. 选择「饼图」→「环形图」
7. 添加数据标签（显示百分比）
8. 设置图例位置
9. 添加图表标题：「各地区销售占比」

### Step 8: 创建联动切片器

**目标**：让多个数据透视表和图表共享切片器

**操作**：
1. 选中任意切片器
2. 「切片器工具」选项卡 → 「报表连接」
3. 在弹出的对话框中勾选所有数据透视表
4. 点击「确定」

**效果说明**：
- 调整切片器时，所有关联的数据透视表和图表都会同步更新
- 实现交互式数据探索

## 最终效果

完成后的数据表应包含：
- 基础数据透视表：按地区和产品类别分析销售额和利润
- 月份切片器：时间筛选
- 数据透视图：柱状图、折线图、环形图
- 利润率计算字段：分析各产品类别的盈利能力
- 联动切片器：所有图表共享筛选

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 数据透视图不更新 | 切片器未连接 | 使用「报表连接」功能 |
| 计算字段显示错误 | 公式引用错误 | 检查公式正确性 |
| 图表标签重叠 | 数据太多 | 调整图表大小或使用其他图表类型 |
| 切片器样式不美观 | 默认样式 | 在「切片器工具」中选择合适样式 |
