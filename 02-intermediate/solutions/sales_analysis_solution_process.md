# sales_analysis_raw.xlsx 练习解决方案

## 练习目标

1. 创建数据透视表分析销售数据
2. 使用切片器筛选数据
3. 创建销售趋势图和占比饼图
4. 使用公式计算汇总数据

## 数据说明

- **列A（产品）**：产品名称
- **列B（区域）**：销售区域
- **列C（销售额）**：销售金额
- **列D（日期）**：销售日期
- **列E（销售员）**：销售员姓名

## 解决方案步骤

### Step 1: 创建数据透视表

**目标**：分析各区域各产品的销售情况

**操作**：
1. 选中数据区域（A1:E101）
2. 「插入」选项卡 → 「数据透视表」
3. 在弹出的对话框中确认数据区域
4. 选择放置位置：新建工作表
5. 点击「确定」

### Step 2: 配置数据透视表字段

**目标**：设置行、列、值字段

**操作**：
1. 将「区域」字段拖到「行」区域
2. 将「产品」字段拖到「列」区域
3. 将「销售额」字段拖到「值」区域
4. Excel默认使用「求和」汇总方式

**结果说明**：
- 行显示各区域
- 列显示各产品
- 值显示各区域各产品的销售额总和

### Step 3: 添加切片器

**目标**：添加日期和销售员切片器

**操作**：
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「插入切片器」
3. 在弹出的对话框中勾选「日期」和「销售员」
4. 点击「确定」

**使用切片器**：
- 点击切片器中的选项进行筛选
- 按住Ctrl键可以多选
- 点击切片器右上角的清除按钮可以清除筛选

### Step 4: 创建计算字段

**目标**：计算各区域各产品的销售占比

**操作**：
1. 选中数据透视表
2. 「数据透视表分析」选项卡 → 「字段、项目和集」→「计算字段」
3. 在弹出的对话框中输入：
   - 名称：销售占比
   - 公式：`=销售额 / SUM(销售额)`
4. 点击「添加」→「确定」

**设置格式**：
1. 右键「销售占比」列的任意单元格
2. 选择「值字段设置」
3. 点击「数字格式」
4. 设置为百分比格式，保留2位小数
5. 点击「确定」

### Step 5: 创建销售趋势图

**目标**：展示月度销售趋势

**操作**：
1. 在原始数据中添加「月份」列
2. 在F1单元格输入「月份」
3. 在F2单元格输入公式：`=TEXT(D2, "yyyy年mm月")`
4. 拖拽填充到F101
5. 选中数据区域（F1:F101和C1:C101）
6. 「插入」选项卡 → 「折线图」→「带数据标记的折线图」
7. 添加图表标题：「月度销售趋势图」
8. 添加数据标签
9. 设置日期格式

### Step 6: 创建销售占比饼图

**目标**：展示各产品销售额占比

**操作**：
1. 使用数据透视表汇总各产品销售额
2. 选中产品和销售额列
3. 「插入」选项卡 → 「饼图」→「饼图」
4. 添加数据标签（显示百分比）
5. 设置图例位置为右侧
6. 添加图表标题：「产品销售占比」

### Step 7: 计算汇总数据

**目标**：使用公式计算关键指标

**操作**：
1. 在H1单元格输入「总销售额」，H2输入公式：`=SUM(C2:C101)`
2. 在H3单元格输入「平均销售额」，H4输入公式：`=AVERAGE(C2:C101)`
3. 在H5单元格输入「最高销售额」，H6输入公式：`=MAX(C2:C101)`
4. 在H7单元格输入「最低销售额」，H8输入公式：`=MIN(C2:C101)`
5. 在H9单元格输入「销售笔数」，H10输入公式：`=COUNT(C2:C101)`

**按区域汇总**：
1. 在J1单元格输入「区域」，K1输入「销售额」
2. 在J2:J8输入各区域名称
3. 在K2单元格输入公式：`=SUMIF(B:B, J2, C:C)`
4. 拖拽填充到K8

## 最终效果

完成后的数据表应包含：
- 数据透视表：按区域和产品汇总销售额
- 切片器：日期和销售员筛选
- 计算字段：销售占比
- 折线图：月度销售趋势
- 饼图：产品销售占比
- 汇总数据：总销售额、平均值、最高最低值等

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 数据透视表不更新 | 源数据变化后未刷新 | 右键透视表 →「刷新」 |
| 日期切片器不显示 | 日期格式不正确 | 设置为日期格式后刷新透视表 |
| 图表数据错误 | 数据源选择错误 | 重新选择数据区域 |
| 计算字段结果错误 | 公式引用错误 | 检查公式正确性 |
