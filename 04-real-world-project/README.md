# 04-实战项目：企业年度经营分析

## 项目背景

你是一家科技公司（智云科技）的数据分析师，公司要求你在年末完成 **2024年度经营分析报告**。你需要整合财务、销售、人力资源和库存四个部门的数据，使用 Excel 专业工具完成数据清洗、建模、分析和可视化，最终向管理层汇报。

## 项目目标

1. **数据整合**：使用 Power Query 合并多个部门的数据源
2. **数据建模**：使用 Power Pivot 建立关系模型，计算关键指标
3. **深度分析**：使用 DAX 度量值进行多维度分析
4. **可视化呈现**：制作专业的年度经营分析仪表板

## 数据说明

### 财务数据（finance_data.xlsx）
| 字段 | 说明 |
|------|------|
| 月份 | 1-12月 |
| 收入科目 | 主营业务收入、其他业务收入等 |
| 收入金额 | 当月收入 |
| 成本科目 | 主营业务成本、管理费用等 |
| 成本金额 | 当月成本 |

### 销售数据（sales_data.xlsx）
| 字段 | 说明 |
|------|------|

| 日期 | 销售日期 |
| 产品ID | 关联产品表 |
| 客户ID | 关联客户表 |
| 区域 | 销售区域 |
| 数量 | 销售数量 |
| 单价 | 产品单价 |

### 人力资源数据（hr_data.xlsx）
| 字段 | 说明 |
|------|------|
| 员工ID | 唯一标识 |

| 字段 | 说明 |

## 分析要求

### 财务分析
- 计算月度收入、成本、利润趋势
- 计算利润率、成本率
- 同比分析（如有历史数据）

### 销售分析
- 各区域销售业绩对比
- 产品销售排行
- 客户贡献度分析（ABC分析）
- 月度销售趋势

### 人力资源分析
- 各部门人员结构
- 薪资分布分析
- 离职率统计
- 绩效分布

### 库存分析
- 库存周转率
- 滞销产品识别
- 库存金额占比

### 综合分析
- 收入-利润联动分析
- 人效分析（人均产值）
- 综合KPI仪表板

## 交付物

1. 清洗后的合并数据表
2. 数据模型（含关系图）
3. 分析报表（多个数据透视表）
4. 年度经营分析仪表板
5. 分析结论与建议（Word或PPT）
