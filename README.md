# Excel 学习路径 - 从基础到专业

这是一个完整的 Excel 学习项目，涵盖从入门到精通的全套学习资料和练习体系。

## 📚 学习路径图

### 🔰 基础阶段 (01-basics)
- **目标**：掌握 Excel 核心界面和基本操作
- **内容**：工作簿/工作表/单元格操作、数据输入与格式化、基础公式与函数
- **文件**：3个教程文档 + 2个练习文件 + 2个答案文件

### 🚀 进阶阶段 (02-intermediate)
- **目标**：精通函数、数据透视表和图表
- **内容**：逻辑与查找函数、文本与日期函数、数据透视表与图表、数据验证与假设分析
- **文件**：4个教程文档 + 3个练习文件 + 3个答案文件

### 🧠 专业阶段 (03-professional)
- **目标**：掌握高级数据分析工具
- **内容**：Power Query 数据清洗、Power Pivot 与 DAX、VBA 宏自动化、仪表板设计
- **文件**：4个教程文档 + 4个练习文件 + 4个答案文件

## 💻 推荐 Excel 版本

| 版本 | 推荐度 | 说明 |
|------|--------|------|
| Microsoft 365 | ⭐⭐⭐⭐⭐ | 最佳选择，包含所有最新功能（动态数组、XLOOKUP等） |
| Excel 2021 | ⭐⭐⭐⭐ | 包含大部分高级功能 |
| Excel 2019 | ⭐⭐⭐ | 部分功能受限（无动态数组） |
| Excel 2016 | ⭐⭐ | 不推荐，缺少多项关键功能 |

## 📖 如何使用本仓库

1. **按顺序学习**：从基础阶段开始，依次完成进阶和专业阶段
2. **先练后看**：打开练习文件尝试完成任务，遇到困难再查看答案
3. **理解原理**：不仅要知道怎么做，还要理解为什么这么做
4. **动手实践**：每个练习文件都配有详细的操作步骤说明
5. **代码生成**：所有练习数据都由 Python 脚本生成，可复现

## 🎯 学习建议

### 基础阶段（1-2周）
- 每天学习一个教程文档
- 完成对应的练习文件
- 掌握常用快捷键（Ctrl+C/V/X, Ctrl+Z/Y, Ctrl+S等）

### 进阶阶段（2-3周）
- 重点掌握查找函数（VLOOKUP/XLOOKUP/INDEX+MATCH）
- 熟练使用数据透视表
- 学会创建各种图表

### 专业阶段（3-4周）
- Power Query：数据清洗的利器
- Power Pivot：数据分析的核心
- VBA：自动化的必备技能
- 仪表板：展示分析成果

## 📝 学习技巧

1. **多练习**：Excel 是工具，熟能生巧
2. **多思考**：遇到问题先思考，再查资料
3. **多总结**：记录常用公式和操作技巧
4. **多分享**：教别人是最好的学习方式
