# 专业阶段 - Power Query、Power Pivot、VBA、仪表板

## 阶段目标

本阶段旨在帮助学习者掌握 Excel 高级数据处理和自动化技能，包括 Power Query 数据清洗、Power Pivot 数据建模、VBA 宏编程和仪表板设计等专业技能。

## 学习内容

1. **Power Query 数据导入与清洗** - 从文件/Web导入、合并查询、数据清洗、M语言基础
2. **Power Pivot 与 DAX 基础** - 加载数据到数据模型、建立关系、DAX度量值
3. **VBA 宏与自动化** - 录制宏、VBA编辑器、基本语法、循环与条件、用户表单
4. **仪表板与报表设计** - 仪表板设计原则、使用图表与KPIs构建综合仪表板

## 适用版本

- Excel 2016+（需要安装 Power Query 插件或使用内置功能）
- Microsoft 365（推荐，所有功能完整支持）

## 学习建议

1. Power Query 和 Power Pivot 是数据分析的利器，建议先掌握这两项技能
2. VBA 需要一定的编程基础，可以从录制宏开始学习
3. 仪表板设计需要综合运用前面所学的所有技能
4. 完成练习后对照答案检查，重点理解操作思路

## 版本注意事项

- Power Query 在 Excel 2016+ 内置，Excel 2013 需要单独安装插件
- Power Pivot 需要在 Excel 选项中启用
- VBA 在所有版本中都可用，但宏安全设置可能阻止运行
