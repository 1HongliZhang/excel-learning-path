# Power Query 数据导入与清洗

## 一、Power Query 基础

### 1.1 什么是 Power Query

Power Query 是 Excel 中的数据连接和转换工具，可以：
- 从多种数据源导入数据
- 清洗和转换数据
- 自动刷新数据
- 合并多个数据源

### 1.2 打开 Power Query

**方法一：数据选项卡**
1. 「数据」选项卡 → 「获取数据」
2. 选择数据源类型

**方法二：快捷键**
- `Alt+A, P, N`：打开新建查询菜单

**方法三：从表格创建**
1. 选中数据区域
2. 「数据」选项卡 → 「从表格/范围」

### 1.3 Power Query 界面

**功能区**：
- **开始**：常用操作（刷新、关闭、合并）
- **转换**：数据转换操作
- **添加列**：添加新列
- **视图**：界面设置

**查询窗格**：
- 显示所有查询
- 可以重命名、删除查询

**数据预览窗口**：
- 显示数据预览
- 可以直接点击列标题筛选

## 二、数据导入

### 2.1 从文件导入

**从 Excel 文件导入**：
1. 「数据」→「获取数据」→「自文件」→「自 Excel 工作簿」
2. 选择 Excel 文件
3. 选择要导入的工作表
4. 点击「加载」或「转换数据」

**从 CSV 文件导入**：
1. 「数据」→「获取数据」→「自文件」→「自 CSV」
2. 选择 CSV 文件
3. 预览数据，设置分隔符
4. 点击「加载」或「转换数据」

### 2.2 从 Web 导入

**操作步骤**：
1. 「数据」→「获取数据」→「自其他来源」→「自 Web」
2. 输入网页 URL
3. 选择要导入的表格
4. 点击「加载」或「转换数据」

**注意事项**：
- 需要网络连接
- 网页结构变化可能导致导入失败
- 部分网站可能有反爬虫机制

### 2.3 从数据库导入

**从 SQL Server 导入**：
1. 「数据」→「获取数据」→「自数据库」→「自 SQL Server 数据库」
2. 输入服务器名称和数据库名称
3. 选择认证方式
4. 选择要导入的表或编写 SQL 查询
5. 点击「加载」或「转换数据」

**从其他数据库导入**：
- 支持 Access、Oracle、MySQL、PostgreSQL 等

## 三、数据清洗

### 3.1 删除列

**操作步骤**：
1. 选中要删除的列（可多选）
2. 右键 →「删除列」
3. 或在「开始」选项卡点击「删除列」

**快速删除**：
- 选中列后按 Delete 键

### 3.2 重命名列

**操作步骤**：
1. 双击列标题
2. 输入新名称
3. 按 Enter 确认

### 3.3 更改数据类型

**操作步骤**：
1. 点击列标题左侧的数据类型图标
2. 选择目标数据类型
3. 或在「转换」选项卡点击「数据类型」

**常用数据类型**：
- 文本
- 整数
- 小数
- 日期/时间
- 布尔值

### 3.4 拆分列

**按分隔符拆分**：
1. 选中要拆分的列
2. 「转换」选项卡 →「拆分列」→「按分隔符」
3. 选择分隔符（逗号、空格、自定义）
4. 设置拆分方式（拆分为列、拆分为行）
5. 点击「确定」

**按位置拆分**：
1. 选中要拆分的列
2. 「转换」→「拆分列」→「按位置」
3. 设置拆分位置
4. 点击「确定」

### 3.5 替换值

**操作步骤**：
1. 选中要替换值的列
2. 「开始」选项卡 →「替换值」
3. 输入要查找的值和替换值
4. 点击「确定」

**高级替换**：
- 可以使用通配符
- 可以基于条件替换

### 3.6 填充缺失值

**操作步骤**：
1. 选中要填充的列
2. 「转换」选项卡 →「填充」
3. 选择填充方式：
   - 向下填充：用上一行的值填充
   - 向上填充：用下一行的值填充
   - 向前填充：用前一个非空值填充
   - 向后填充：用后一个非空值填充

### 3.7 删除重复值

**操作步骤**：
1. 选中要检查重复值的列
2. 「开始」选项卡 →「删除重复项」
3. 确认要检查的列
4. 点击「确定」

### 3.8 逆透视列

**场景**：将宽表转换为长表

**操作步骤**：
1. 选中要逆透视的列
2. 「转换」选项卡 →「逆透视列」→「逆透视其他列」
3. Power Query 会自动创建「属性」和「值」两列

**示例**：

| 姓名 | 1月 | 2月 | 3月 |
|------|-----|-----|-----|
| 张三 | 100 | 200 | 300 |

逆透视后：

| 姓名 | 属性 | 值 |
|------|------|----|
| 张三 | 1月 | 100 |
| 张三 | 2月 | 200 |
| 张三 | 3月 | 300 |

## 四、合并查询

### 4.1 合并查询

**操作步骤**：
1. 「开始」选项卡 →「合并查询」→「合并」
2. 选择第一个查询和第二个查询
3. 选择连接字段
4. 选择连接类型：
   - 左外部：保留第一个查询的所有行
   - 右外部：保留第二个查询的所有行
   - 完全外部：保留两个查询的所有行
   - 内部：只保留匹配的行
5. 点击「确定」

### 4.2 追加查询

**操作步骤**：
1. 「开始」选项卡 →「追加查询」→「追加」
2. 选择要追加的查询
3. 点击「确定」

**注意事项**：
- 追加的查询需要有相同的列结构
- 列顺序可以不同，但列名必须匹配

## 五、M语言基础

### 5.1 什么是 M 语言

M 语言是 Power Query 的查询语言，用于编写数据转换脚本。

**特点**：
- 函数式编程语言
- 大小写敏感
- 强类型

### 5.2 常用 M 函数

**导入数据**：
```powerquery-m
= Excel.CurrentWorkbook(){[Name="表1"]}[Content]
= Csv.Document(File.Contents("data.csv"))
= Web.Page(Web.Contents("https://example.com"))
```

**数据转换**：
```powerquery-m
= Table.SelectColumns(源, {"姓名", "部门", "薪资"})
= Table.RenameColumns(源, {{"旧名称", "新名称"}})
= Table.TransformColumnTypes(源, {{"薪资", Int64.Type}})
= Table.SplitColumn(源, "列名", Splitter.SplitTextByDelimiter(","))
= Table.ReplaceValue(源, "旧值", "新值", Replacer.ReplaceText, {"列名"})
```

**合并查询**：
```powerquery-m
= Table.NestedJoin(表1, {"ID"}, 表2, {"ID"}, "表2", JoinKind.LeftOuter)
= Table.Combine({表1, 表2})
```

### 5.3 高级编辑器

**打开高级编辑器**：
1. 在 Power Query 界面中点击「高级编辑器」
2. 可以查看和编辑 M 代码
3. 点击「完成」保存更改

## 六、示例练习

### 6.1 清洗销售数据

**场景**：清洗包含错误格式和缺失值的销售数据

**操作步骤**：
1. 导入数据到 Power Query
2. 删除不需要的列
3. 更改数据类型（日期、数值）
4. 填充缺失值
5. 拆分包含多个信息的列
6. 删除重复行
7. 加载清洗后的数据到 Excel

### 6.2 合并多表数据

**场景**：合并员工表和部门表

**操作步骤**：
1. 导入员工表和部门表
2. 使用「合并查询」功能
3. 选择员工表的「部门ID」和部门表的「ID」作为连接字段
4. 选择左外部连接
5. 展开合并的列
6. 加载合并后的数据

### 6.3 逆透视数据

**场景**：将月度销售数据从宽表转换为长表

**操作步骤**：
1. 导入宽表数据
2. 选中日期列以外的所有列
3. 「转换」→「逆透视列」→「逆透视其他列」
4. 重命名「属性」列为「月份」
5. 重命名「值」列为「销售额」
6. 加载转换后的数据

## 七、常见错误与技巧

### 7.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 数据类型错误 | 列数据类型不匹配 | 检查并更改数据类型 |
| 合并失败 | 连接字段数据类型不匹配 | 确保连接字段类型一致 |
| 查询无法刷新 | 数据源路径变化 | 更新数据源路径 |
| 内存不足 | 数据量过大 | 过滤不必要的数据 |

### 7.2 实用技巧

1. **快速刷新查询**：右键查询 →「刷新」
2. **禁用加载**：在查询设置中取消勾选「启用加载」
3. **创建参数查询**：使用参数动态控制查询
4. **分组依据**：使用分组功能汇总数据
