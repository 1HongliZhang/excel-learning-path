# 仪表板与报表设计

## 一、仪表板设计原则

### 1.1 仪表板概述

仪表板是数据可视化的综合展示，用于：
- 实时监控关键业务指标
- 快速发现问题和趋势
- 支持决策制定

### 1.2 设计原则

**1. 清晰简洁**
- 只展示最重要的信息
- 避免信息过载
- 使用清晰的标题和标签

**2. 层次分明**
- 将相关信息分组
- 使用空白分隔不同区域
- 建立视觉层次

**3. 视觉美观**
- 使用协调的配色方案
- 选择合适的图表类型
- 保持一致性

**4. 交互性**
- 使用切片器进行筛选
- 添加数据标签
- 支持钻取分析

### 1.3 信息分类

**KPI（关键绩效指标）**：
- 核心指标（总销售额、利润率等）
- 使用大字体突出显示
- 显示同比/环比变化

**趋势分析**：
- 使用折线图展示变化趋势
- 显示预测线
- 标注关键事件

**对比分析**：
- 使用柱状图进行对比
- 显示占比关系
- 突出异常值

**明细数据**：
- 使用表格展示详细数据
- 支持排序和筛选
- 提供导出功能

## 二、仪表板布局设计

### 2.1 布局类型

**顶部-内容布局**：
- 顶部：标题和全局筛选器
- 中间：核心KPI和主要图表
- 底部：明细数据

**左右布局**：
- 左侧：筛选器和导航
- 右侧：主要内容

**网格布局**：
- 将仪表板分成多个区域
- 每个区域展示一个图表或指标

### 2.2 布局工具

**Excel 表格功能**：
- 使用表格组织内容
- 合并单元格创建区域
- 使用边框分隔不同区域

**形状和线条**：
- 插入形状作为背景
- 使用线条分隔区域
- 添加标题框

**切片器**：
- 添加筛选器进行交互式筛选
- 调整切片器样式和位置
- 连接多个图表

## 三、图表选择与设计

### 3.1 图表类型选择

| 数据类型 | 推荐图表 | 说明 |
|----------|----------|------|
| 趋势变化 | 折线图 | 展示数据随时间变化 |
| 类别对比 | 柱状图/条形图 | 比较不同类别的数值 |
| 占比关系 | 饼图/环形图 | 展示各部分占总体的比例 |
| 排名 | 水平条形图 | 显示排名顺序 |
| 相关性 | 散点图 | 展示两个变量之间的关系 |
| 分布 | 直方图 | 展示数据分布情况 |

### 3.2 图表美化技巧

**1. 去除冗余元素**：
- 删除不必要的网格线
- 简化图例
- 去除边框

**2. 使用专业配色**：
- 使用公司品牌色
- 避免过多颜色
- 使用渐变色增加层次感

**3. 添加数据标签**：
- 在图表上直接显示数值
- 使用数据标签代替图例
- 格式化数据标签

**4. 强调重点**：
- 使用不同颜色突出重点数据
- 添加注释和标记
- 使用箭头指向关键数据点

### 3.3 创建动态图表

**使用切片器**：
1. 创建数据透视表
2. 插入切片器
3. 基于透视表创建图表
4. 图表会随切片器更新

**使用公式**：
1. 使用 OFFSET 函数创建动态范围
2. 基于动态范围创建图表
3. 图表会自动更新

## 四、KPI 设计

### 4.1 KPI 卡片设计

**创建 KPI 卡片**：
1. 插入形状（矩形）
2. 在形状内添加文本框
3. 输入 KPI 名称和数值
4. 设置形状样式
5. 添加趋势箭头

**示例代码（VBA）**：
```vba
Sub UpdateKPI()
    Dim ws As Worksheet
    Set ws = Worksheets("仪表板")
    
    ws.Range("B2").Value = Format(Sheet2.Range("K1").Value, "¥#,##0")
    ws.Range("B3").Value = "同比增长 " & Format(Sheet2.Range("K2").Value, "+0.00%")
    
    If Sheet2.Range("K2").Value > 0 Then
        ws.Range("B3").Font.Color = RGB(0, 153, 0)
    Else
        ws.Range("B3").Font.Color = RGB(255, 0, 0)
    End If
End Sub
```

### 4.2 KPI 计算

**常用 KPI**：
```dax
总销售额 := SUM(销售表[销售额])
总利润 := SUM(销售表[利润])
利润率 := DIVIDE([总利润], [总销售额])
销售数量 := COUNT(销售表[订单ID])
客户数量 := DISTINCTCOUNT(销售表[客户ID])
平均订单金额 := DIVIDE([总销售额], [销售数量])
```

**同比分析**：
```dax
去年同期销售额 := CALCULATE(
    SUM(销售表[销售额]),
    SAMEPERIODLASTYEAR(日期表[日期])
)
同比增长率 := DIVIDE(
    [总销售额] - [去年同期销售额],
    [去年同期销售额]
)
```

## 六、保护工作表与工作簿

### 6.1 保护工作表

**操作步骤**：
1. 选中要保护的工作表
2. 「审阅」选项卡 →「保护工作表」
3. 在弹出的对话框中：
   - 设置密码（可选）
   - 勾选允许用户进行的操作
   - 点击「确定」
4. 再次输入密码确认

**允许的操作**：
- 选定锁定单元格
- 选定未锁定单元格
- 编辑对象
- 编辑方案

### 6.2 保护工作簿

**操作步骤**：
1. 「审阅」选项卡 →「保护工作簿」
2. 在弹出的对话框中：
   - 设置密码（可选）
   - 勾选保护结构和窗口
   - 点击「确定」
3. 再次输入密码确认

**保护内容**：
- 工作表结构（防止删除、重命名、移动）
- 窗口大小和位置

### 6.3 取消保护

**操作步骤**：
1. 「审阅」选项卡 →「取消保护工作表」或「取消保护工作簿」
2. 输入密码（如果设置了密码）
3. 点击「确定」

## 七、常见错误与技巧

### 7.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 图表数据错误 | 数据源选择错误 | 重新选择数据区域 |
| 仪表板刷新慢 | 数据量过大 | 使用数据模型或筛选数据 |
| 切片器不联动 | 未连接所有图表 | 使用报表连接功能 |
| 布局混乱 | 未使用表格或形状 | 使用表格和形状组织布局 |

### 7.2 实用技巧

1. **使用主题**：选择合适的 Excel 主题统一风格
2. **使用对齐工具**：确保元素对齐
3. **使用网格线**：辅助布局设计
4. **测试交互性**：确保所有筛选器正常工作
5. **添加刷新按钮**：方便用户刷新数据
