# Power Pivot 与 DAX 基础

## 一、Power Pivot 基础

### 1.1 什么是 Power Pivot

Power Pivot 是 Excel 的数据建模工具，可以：
- 创建数据模型（类似数据库）
- 建立表之间的关系
- 使用 DAX 编写度量值
- 创建复杂的数据分析报表

### 1.2 启用 Power Pivot

**操作步骤**：
1. 「文件」选项卡 →「选项」→「加载项」
2. 在「管理」下拉菜单中选择「COM 加载项」
3. 点击「转到」
4. 勾选「Microsoft Power Pivot for Excel」
5. 点击「确定」

**验证启用**：
- 检查是否出现「Power Pivot」选项卡

### 1.3 加载数据到数据模型

**方法一：从表格加载**
1. 选中数据区域
2. 「插入」选项卡 →「表格」
3. 确认数据区域，勾选「表包含标题」
4. 点击「确定」
5. 「表格设计」选项卡 →「添加到数据模型」

**方法二：从 Power Query 加载**
1. 在 Power Query 中完成数据清洗
2. 点击「关闭并上载至」→「仅创建连接」
3. 勾选「添加此数据至数据模型」
4. 点击「确定」

**方法三：直接导入**
1. 「Power Pivot」选项卡 →「管理」
2. 在 Power Pivot 窗口中点击「从其他源」
3. 选择数据源类型
4. 按照向导完成导入

## 二、建立关系

### 2.1 关系概述

关系是表之间的连接，基于共同的字段。

**类型**：
- 一对一关系
- 一对多关系（最常见）
- 多对多关系

### 2.2 创建关系

**操作步骤**：
1. 打开 Power Pivot 窗口
2. 「设计」选项卡 →「关系图」
3. 拖拽一个表的字段到另一个表的对应字段
4. 释放鼠标，关系自动创建

**验证关系**：
- 在关系图中会显示连接线
- 可以双击连接线编辑关系

### 2.3 编辑关系

**操作步骤**：
1. 双击关系连接线
2. 在弹出的对话框中：
   - 选择左表和右表
   - 选择连接字段
   - 设置交叉筛选方向（单向/双向）
   - 设置 Cardinality（一对一/一对多）
3. 点击「确定」

## 三、DAX 基础

### 3.1 什么是 DAX

DAX（Data Analysis Expressions）是 Power Pivot 和 Power BI 的公式语言。

**特点**：
- 类似 Excel 公式
- 函数式语言
- 支持上下文

### 3.2 DAX 与 Excel 公式的区别

| 特性 | Excel 公式 | DAX |
|------|-----------|-----|
| 数据模型 | 基于单元格 | 基于表和列 |
| 聚合 | 需要手动编写 | 自动处理 |
| 上下文 | 单元格上下文 | 行上下文和筛选上下文 |
| 函数数量 | 约 400 个 | 约 200 个（不断增加） |

### 3.3 常用 DAX 函数

**聚合函数**：
```dax
SUM(表名[列名])           // 求和
AVERAGE(表名[列名])       // 平均值
COUNT(表名[列名])         // 计数（仅数字）
COUNTA(表名[列名])        // 计数（非空）
MIN(表名[列名])           // 最小值
MAX(表名[列名])           // 最大值
DISTINCTCOUNT(表名[列名]) // 去重计数
```

**筛选函数**：
```dax
CALCULATE(表达式, 筛选条件1, 筛选条件2, ...)  // 修改筛选上下文
CALCULATETABLE(表达式, 筛选条件1, ...)        // 返回表
FILTER(表, 条件)                              // 筛选表
ALL(表名)                                     // 移除所有筛选
ALLSELECTED(表名)                             // 保留当前筛选
RELATED(表名[列名])                            // 获取关联表的值
```

**时间智能函数**：
```dax
TOTALYTD(表达式, 日期列)          // 年初至今
TOTALQTD(表达式, 日期列)          // 季度初至今
TOTALMTD(表达式, 日期列)          // 月初至今
SAMEPERIODLASTYEAR(日期列)        // 去年同期
DATESYTD(日期列)                  // 返回年初至今的日期表
DATEADD(日期列, 天数, DAY)        // 日期加减
```

**排名函数**：
```dax
RANKX(表, 表达式, [值], [升序], [排名方式])  // 排名
RANK.EQ(值, 列)                             // Excel 兼容排名
RANK.AVG(值, 列)                             // 平均排名
```

### 3.4 创建度量值

**操作步骤**：
1. 打开 Power Pivot 窗口
2. 选中要添加度量值的表
3. 在公式栏中输入：
   ```dax
   度量值名称 := 表达式
   ```
4. 按 Enter 确认

**示例**：
```dax
总销售额 := SUM(销售表[销售额])
平均销售额 := AVERAGE(销售表[销售额])
销售数量 := COUNT(销售表[订单ID])
```

### 3.5 行上下文与筛选上下文

**行上下文**：
- 计算每行时的上下文
- 使用 `CALCULATE` 可以转换为筛选上下文

**筛选上下文**：
- 当前筛选条件下的上下文
- 由切片器、筛选器、行/列标签决定

**示例**：
```dax
销售额占比 := DIVIDE(
    SUM(销售表[销售额]),
    CALCULATE(SUM(销售表[销售额]), ALL(销售表))
)
```

## 四、示例练习

### 4.1 创建基础度量值

**场景**：创建销售分析度量值

**操作步骤**：
1. 打开 Power Pivot 窗口
2. 在「销售表」中创建以下度量值：
   ```dax
   总销售额 := SUM(销售表[销售额])
   总利润 := SUM(销售表[利润])
   利润率 := DIVIDE([总利润], [总销售额])
   销售数量 := COUNT(销售表[订单ID])
   平均单价 := DIVIDE([总销售额], [销售数量])
   ```

### 4.2 创建时间智能度量值

**场景**：创建同比分析度量值

**操作步骤**：
1. 确保有日期表并建立关系
2. 创建以下度量值：
   ```dax
   年初至今销售额 := TOTALYTD(SUM(销售表[销售额]), 日期表[日期])
   去年同期销售额 := CALCULATE(
       SUM(销售表[销售额]),
       SAMEPERIODLASTYEAR(日期表[日期])
   )
   同比增长率 := DIVIDE(
       [总销售额] - [去年同期销售额],
       [去年同期销售额]
   )
   ```

### 4.3 创建排名度量值

**场景**：创建产品销售排名

**操作步骤**：
1. 创建以下度量值：
   ```dax
   产品销售排名 := RANKX(
       ALL(产品表[产品名称]),
       CALCULATE(SUM(销售表[销售额]))
   )
   区域销售排名 := RANKX(
       ALL(区域表[区域名称]),
       CALCULATE(SUM(销售表[销售额]))
   )
   ```

### 4.4 使用 CALCULATE 创建条件度量值

**场景**：创建特定条件的度量值

**操作步骤**：
1. 创建以下度量值：
   ```dax
   华北区域销售额 := CALCULATE(
       SUM(销售表[销售额]),
       区域表[区域名称] = "华北"
   )
   电子产品销售额 := CALCULATE(
       SUM(销售表[销售额]),
       产品表[类别] = "电子产品"
   )
   2024年销售额 := CALCULATE(
       SUM(销售表[销售额]),
       YEAR(日期表[日期]) = 2024
   )
   ```

## 五、数据透视表与数据模型

### 5.1 使用数据模型创建数据透视表

**操作步骤**：
1. 「插入」选项卡 →「数据透视表」
2. 在弹出的对话框中选择「使用此工作簿的数据模型」
3. 选择放置位置
4. 点击「确定」

**字段列表**：
- 显示所有表及其字段
- 可以跨表选择字段
- 度量值会显示在「度量值」文件夹中

### 5.2 添加计算列

**操作步骤**：
1. 打开 Power Pivot 窗口
2. 选中要添加计算列的表
3. 在公式栏中输入公式：
   ```dax
   = 表达式
   ```
4. 按 Enter 确认

**示例**：
```dax
= 销售表[销售额] - 销售表[成本]  // 计算利润
= YEAR(日期表[日期])            // 提取年份
= MONTH(日期表[日期])           // 提取月份
```

## 六、常见错误与技巧

### 6.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 度量值返回错误 | 公式语法错误 | 检查公式语法 |
| 关系未建立 | 表之间没有连接 | 建立正确的关系 |
| 时间智能函数错误 | 缺少日期表 | 创建日期表并建立关系 |
| CALCULATE 结果不正确 | 筛选上下文问题 | 检查筛选条件 |

### 6.2 实用技巧

1. **创建日期表**：使用 Power Query 生成连续日期
2. **使用 ALL 函数**：清除筛选上下文
3. **使用 RELATED 函数**：获取关联表的值
4. **使用 DIVIDE 函数**：安全的除法运算（避免除零错误）
