# dashboard_data.xlsx 练习解决方案

## 练习目标

1. 设计仪表板布局
2. 创建核心 KPI 指标
3. 创建交互式图表
4. 添加切片器进行筛选
5. 美化仪表板

## 数据说明

该文件包含销售数据：
- 月份：1月-12月
- 产品类别：电子产品、服装鞋帽、食品饮料、家居用品、办公用品
- 地区：北京、上海、广州、深圳、成都、杭州
- 销售额：销售金额
- 成本：成本金额
- 利润：利润金额

## 解决方案步骤

### Step 1: 创建数据透视表

**操作步骤**：
1. 选中数据区域（A1:F361）
2. 「插入」选项卡 →「数据透视表」
3. 选择放置位置：新建工作表
4. 点击「确定」
5. 配置字段：
   - 行：产品类别、地区
   - 值：销售额、成本、利润

### Step 2: 创建 KPI 卡片

**操作步骤**：
1. 新建工作表，命名为「仪表板」
2. 使用形状创建 KPI 卡片：
   - 插入矩形形状
   - 设置形状样式（填充色、边框）
   - 在形状内添加文本框
   - 输入 KPI 名称和数值

**KPI 公式**：
- 总销售额：`=SUM(数据!D2:D361)`
- 总成本：`=SUM(数据!E2:E361)`
- 总利润：`=SUM(数据!F2:F361)`
- 利润率：`=总利润/总销售额`

**格式化 KPI**：
- 设置大字体（24-32号）
- 设置货币格式（¥#,##0）
- 设置百分比格式（0.00%）
- 使用颜色区分正负增长

### Step 3: 创建销售趋势图

**操作步骤**：
1. 创建新的数据透视表：
   - 行：月份
   - 值：销售额
2. 选中数据透视表
3. 「分析」选项卡 →「数据透视图」
4. 选择「折线图」→「带数据标记的折线图」
5. 添加图表标题：「月度销售趋势」
6. 添加数据标签
7. 设置日期格式
8. 去除网格线

### Step 4: 创建区域销售对比图

**操作步骤**：
1. 创建新的数据透视表：
   - 行：地区
   - 值：销售额
2. 选中数据透视表
3. 「分析」选项卡 →「数据透视图」
4. 选择「柱形图」→「簇状柱形图」
5. 添加图表标题：「各地区销售对比」
6. 添加数据标签
7. 调整图例位置

### Step 5: 创建产品类别占比图

**操作步骤**：
1. 创建新的数据透视表：
   - 行：产品类别
   - 值：销售额
2. 选中数据透视表
3. 「分析」选项卡 →「数据透视图」
4. 选择「饼图」→「环形图」
5. 添加数据标签（显示百分比）
6. 设置图例位置
7. 添加图表标题：「产品类别销售占比」

### Step 6: 添加切片器

**操作步骤**：
1. 选中任意数据透视表
2. 「数据透视表分析」选项卡 →「插入切片器」
3. 勾选「月份」和「产品类别」
4. 点击「确定」
5. 调整切片器位置和样式
6. 将切片器连接到所有数据透视表：
   - 选中切片器
   - 「切片器工具」→「报表连接」
   - 勾选所有数据透视表
   - 点击「确定」

### Step 7: 设计仪表板布局

**布局方案**：
- 顶部：标题和切片器
- 中间左侧：KPI 卡片（总销售额、总成本、总利润、利润率）
- 中间右侧：销售趋势图
- 底部左侧：区域销售对比图
- 底部右侧：产品类别占比图

**操作步骤**：
1. 移动所有图表和切片器到「仪表板」工作表
2. 使用对齐工具对齐元素
3. 调整图表大小和位置
4. 添加背景形状和边框
5. 设置统一的配色方案

### Step 8: 美化仪表板

**操作步骤**：
1. 选择合适的 Excel 主题
2. 统一图表颜色
3. 添加标题和副标题
4. 设置字体和字号
5. 添加公司 Logo（可选）
6. 添加数据来源说明
7. 设置刷新按钮

### Step 9: 保护工作表

**操作步骤**：
1. 「审阅」选项卡 →「保护工作表」
2. 设置密码（可选）
3. 勾选允许用户进行的操作：
   - 选定锁定单元格
   - 选定未锁定单元格
   - 使用切片器
4. 点击「确定」

## 最终效果

完成后的仪表板应包含：
- 顶部标题和筛选器（月份、产品类别）
- 核心 KPI 卡片（总销售额、总成本、总利润、利润率）
- 月度销售趋势折线图
- 各地区销售对比柱状图
- 产品类别销售占比环形图
- 统一的配色方案和样式
- 交互式筛选功能

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 图表数据错误 | 数据源选择错误 | 重新选择数据区域 |
| 仪表板刷新慢 | 数据量过大 | 使用数据模型或筛选数据 |
| 切片器不联动 | 未连接所有图表 | 使用报表连接功能 |
| 布局混乱 | 未使用表格或形状 | 使用表格和形状组织布局 |
| 图表重叠 | 元素位置冲突 | 调整图表大小和位置 |

## 设计技巧

1. **使用主题**：选择合适的 Excel 主题统一风格
2. **使用对齐工具**：确保元素对齐
3. **使用网格线**：辅助布局设计
4. **测试交互性**：确保所有筛选器正常工作
5. **添加刷新按钮**：方便用户刷新数据
6. **保持简洁**：只展示最重要的信息
