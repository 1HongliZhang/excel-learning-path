# messy_data_for_power_query.xlsx 练习解决方案

## 练习目标

1. 使用 Power Query 导入多个工作表数据
2. 清洗数据（统一列名、转换日期格式）
3. 合并多个查询
4. 加载清洗后的数据到 Excel

## 数据说明

该文件包含两个工作表：

**工作表1：上半年数据**
- 日期：格式为 `YYYY-MM-DD`
- 产品名称：产品名称
- 销售区域：销售区域
- 销售额(元)：销售金额

**工作表2：下半年数据**
- 销售日期：格式为 `YYYY年MM月DD日`
- 产品：产品名称
- 区域：销售区域
- 金额：销售金额
- 利润：利润金额

**问题**：
- 两个工作表的列名不一致
- 日期格式不同
- 需要合并为一个统一的表格

## 解决方案步骤

### Step 1: 导入数据到 Power Query

**操作步骤**：
1. 打开 Excel 文件
2. 「数据」选项卡 →「获取数据」→「自文件」→「自 Excel 工作簿」
3. 选择当前文件
4. 在导航器中选择「上半年数据」和「下半年数据」
5. 点击「转换数据」

### Step 2: 清洗「上半年数据」

**操作步骤**：
1. 重命名查询为「上半年数据」
2. 重命名列：
   - 「日期」→「日期」
   - 「产品名称」→「产品名称」
   - 「销售区域」→「区域」
   - 「销售额(元)」→「销售额」
3. 更改数据类型：
   - 日期：日期/时间
   - 产品名称：文本
   - 区域：文本
   - 销售额：整数
4. 添加「利润」列：
   - 「添加列」→「自定义列」
   - 列名：利润
   - 公式：`[销售额] * 0.2`（假设利润率为20%）

### Step 3: 清洗「下半年数据」

**操作步骤**：
1. 重命名查询为「下半年数据」
2. 重命名列：
   - 「销售日期」→「日期」
   - 「产品」→「产品名称」
   - 「区域」→「区域」
   - 「金额」→「销售额」
   - 「利润」→「利润」
3. 更改数据类型：
   - 日期：日期/时间（Excel会自动识别中文日期格式）
   - 产品名称：文本
   - 区域：文本
   - 销售额：整数
   - 利润：小数
4. 如果日期格式无法自动识别：
   - 使用「转换」→「格式」→「从文本转换」→「使用区域设置」

### Step 4: 合并查询

**操作步骤**：
1. 选择「上半年数据」查询
2. 「开始」选项卡 →「追加查询」→「追加查询作为新查询」
3. 在弹出的对话框中：
   - 选择「两个表」
   - 第一个表：上半年数据
   - 第二个表：下半年数据
4. 点击「确定」
5. 重命名新查询为「合并数据」

### Step 5: 数据质量检查

**操作步骤**：
1. 检查是否有空值
2. 检查数据类型是否正确
3. 检查日期范围是否完整
4. 如果有重复数据，使用「删除重复项」

### Step 6: 加载数据到 Excel

**操作步骤**：
1. 在 Power Query 中点击「关闭并上载」
2. Excel 会创建一个新工作表，包含合并后的干净数据
3. 将工作表重命名为「清洗后数据」

## 最终效果

完成后的数据表应包含：
- 统一的列名：日期、产品名称、区域、销售额、利润
- 统一的日期格式
- 完整的全年数据（60行）
- 正确的数据类型

## M 代码参考

**上半年数据查询：**
```powerquery-m
let
    源 = Excel.CurrentWorkbook(){[Name="上半年数据"]}[Content],
    重命名列 = Table.RenameColumns(源, {{"日期", "日期"}, {"产品名称", "产品名称"}, {"销售区域", "区域"}, {"销售额(元)", "销售额"}}),
    更改类型 = Table.TransformColumnTypes(重命名列, {{"日期", type date}, {"产品名称", type text}, {"区域", type text}, {"销售额", Int64.Type}}),
    添加利润列 = Table.AddColumn(更改类型, "利润", each [销售额] * 0.2, type number)
in
    添加利润列
```

**合并数据查询：**
```powerquery-m
let
    源 = Table.Combine({上半年数据, 下半年数据})
in
    源
```

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 日期格式无法识别 | 日期格式不一致 | 使用文本转换功能 |
| 合并后列顺序混乱 | 列名不完全匹配 | 确保两个查询的列名完全一致 |
| 数据类型错误 | 原始数据格式问题 | 手动更改数据类型 |
| 查询无法刷新 | 源文件位置变化 | 更新数据源路径 |
