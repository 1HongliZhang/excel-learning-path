# sales_model_for_power_pivot.xlsx 练习解决方案

## 练习目标

1. 加载数据到数据模型
2. 建立表之间的关系
3. 使用 DAX 编写度量值
4. 创建数据透视表展示模型

## 数据说明

该文件包含两个工作表：

**产品表**：
- 产品ID：唯一标识
- 产品名称：产品名称
- 类别：产品类别
- 成本：产品成本
- 售价：产品售价

**销售表**：
- 订单ID：订单编号
- 日期：销售日期
- 产品ID：关联产品表
- 区域：销售区域
- 数量：销售数量
- 销售员：销售员姓名

## 解决方案步骤

### Step 1: 加载数据到数据模型

**操作步骤**：
1. 打开 Excel 文件
2. 选中「产品表」数据区域
3. 「插入」选项卡 →「表格」
4. 确认数据区域，勾选「表包含标题」
5. 点击「确定」
6. 「表格设计」选项卡 →「添加到数据模型」
7. 对「销售表」重复以上步骤

### Step 2: 建立关系

**操作步骤**：
1. 「Power Pivot」选项卡 →「管理」
2. 在 Power Pivot 窗口中点击「关系图」
3. 拖拽「产品表」的「产品ID」字段到「销售表」的「产品ID」字段
4. 释放鼠标，关系自动创建
5. 验证关系：双击连接线，确认连接类型为「一对多」

### Step 3: 创建日期表

**操作步骤**：
1. 在 Power Pivot 窗口中点击「设计」→「新建表」
2. 在公式栏中输入：
   ```dax
   = CALENDAR(DATE(2024,1,1), DATE(2024,12,31))
   ```
3. 按 Enter 确认
4. 重命名表为「日期表」
5. 添加计算列：
   - 年份：`= YEAR([Date])`
   - 月份：`= MONTH([Date])`
   - 季度：`= CEILING([月份]/3, 1)`
6. 建立「日期表」和「销售表」的关系：
   - 拖拽「日期表」的「Date」字段到「销售表」的「日期」字段

### Step 4: 创建基础度量值

**操作步骤**：
1. 在 Power Pivot 窗口中选中「销售表」
2. 在公式栏中创建以下度量值：

**总销售额**：
```dax
总销售额 := SUMX(销售表, 销售表[数量] * RELATED(产品表[售价]))
```

**总成本**：
```dax
总成本 := SUMX(销售表, 销售表[数量] * RELATED(产品表[成本]))
```

**总利润**：
```dax
总利润 := [总销售额] - [总成本]
```

**利润率**：
```dax
利润率 := DIVIDE([总利润], [总销售额])
```

**销售数量**：
```dax
销售数量 := SUM(销售表[数量])
```

**订单数量**：
```dax
订单数量 := DISTINCTCOUNT(销售表[订单ID])
```

### Step 5: 创建时间智能度量值

**操作步骤**：
1. 创建以下度量值：

**年初至今销售额**：
```dax
年初至今销售额 := TOTALYTD([总销售额], 日期表[Date])
```

**去年同期销售额**：
```dax
去年同期销售额 := CALCULATE([总销售额], SAMEPERIODLASTYEAR(日期表[Date]))
```

**同比增长率**：
```dax
同比增长率 := DIVIDE([总销售额] - [去年同期销售额], [去年同期销售额])
```

### Step 6: 创建排名度量值

**操作步骤**：
1. 创建以下度量值：

**产品销售排名**：
```dax
产品销售排名 := RANKX(ALL(产品表[产品名称]), CALCULATE([总销售额]))
```

**区域销售排名**：
```dax
区域销售排名 := RANKX(ALL(销售表[区域]), CALCULATE([总销售额]))
```

### Step 7: 创建数据透视表

**操作步骤**：
1. 「插入」选项卡 →「数据透视表」
2. 选择「使用此工作簿的数据模型」
3. 选择放置位置：新建工作表
4. 点击「确定」
5. 配置字段：
   - 行：区域、产品名称
   - 值：总销售额、总利润、利润率、销售数量
6. 添加切片器：日期（年份、季度、月份）

### Step 8: 创建数据透视图

**操作步骤**：
1. 选中数据透视表
2. 「分析」选项卡 →「数据透视图」
3. 选择图表类型（如柱状图、折线图）
4. 添加图表标题
5. 添加数据标签

## 最终效果

完成后的数据模型应包含：
- 三个表：产品表、销售表、日期表
- 两个关系：产品表-销售表（一对多）、日期表-销售表（一对多）
- 多个度量值：总销售额、总成本、总利润、利润率、同比增长率等
- 数据透视表：按区域和产品分析销售数据
- 切片器：日期筛选

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 度量值返回错误 | 公式语法错误 | 检查公式语法 |
| 关系未建立 | 表之间没有连接 | 建立正确的关系 |
| 时间智能函数错误 | 缺少日期表 | 创建日期表并建立关系 |
| CALCULATE 结果不正确 | 筛选上下文问题 | 检查筛选条件 |
| RELATED 函数错误 | 关系未建立或方向错误 | 检查关系设置 |
