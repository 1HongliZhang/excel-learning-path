# 数据录入与格式化

## 一、快速填充技巧

### 1.1 自动填充（AutoFill）

**填充序列：**
1. 在第一个单元格输入起始值（如 1、2024/1/1、周一）
2. 鼠标移到单元格右下角，当指针变为「+」时拖拽
3. Excel 会自动识别序列规律并填充

**常见序列类型：**
- 数字序列：1, 2, 3, ...
- 日期序列：2024/1/1, 2024/1/2, ...
- 星期序列：周一, 周二, 周三, ...
- 月份序列：一月, 二月, 三月, ...

**自定义序列：**
1. 「文件」→「选项」→「高级」→「编辑自定义列表」
2. 输入自定义序列，每项占一行
3. 点击「添加」

### 1.2 快速填充（Flash Fill）

快速填充可以智能识别数据模式，自动填充数据。

**操作步骤：**
1. 在相邻列手动输入第一个期望结果
2. 选中该单元格
3. 「数据」选项卡 → 「快速填充」（或按 `Ctrl+E`）

**示例场景：**
- 从「张三-销售部-北京」提取姓名：输入「张三」→ `Ctrl+E`
- 将「20240101」转换为日期格式：输入「2024/1/1」→ `Ctrl+E`
- 拆分「zhang_san」为姓和名

## 二、数据格式设置

### 2.1 数字格式

**设置方法：**
1. 选中单元格或区域
2. 右键 → 「设置单元格格式」（或按 `Ctrl+1`）
3. 在「数字」选项卡中选择格式类型

**常用数字格式：**

| 格式类型 | 说明 | 示例 |
|----------|------|------|
| 常规 | 默认格式，不特别指定 | 12345 |
| 数值 | 可设置小数位数、千位分隔符 | 12,345.00 |
| 货币 | 添加货币符号 | ¥12,345.00 |
| 会计专用 | 货币符号左对齐，数值右对齐 | ¥   12,345.00 |
| 百分比 | 乘以100并添加%符号 | 95% |
| 分数 | 以分数形式显示 | 1/2 |
| 科学记数 | 科学计数法 | 1.23E+04 |

### 2.2 日期格式

**常用日期格式：**

| 格式代码 | 显示效果 |
|----------|----------|
| yyyy/mm/dd | 2024/01/15 |
| yyyy年m月d日 | 2024年1月15日 |
| mm-dd-yyyy | 01-15-2024 |
| dddd | 星期一 |

### 2.3 文本格式

当需要将数字作为文本处理时（如身份证号、电话号码）：

**方法一：**
1. 在输入数字前先输入单引号 `'`
2. 例如：`'123456789012345678`

**方法二：**
1. 选中单元格
2. 设置单元格格式为「文本」
3. 再输入数字

## 三、条件格式

条件格式可以根据单元格内容自动应用不同的格式。

### 3.1 创建条件格式

**操作步骤：**
1. 选中要设置条件格式的区域
2. 「开始」选项卡 → 「条件格式」
3. 选择条件类型

### 3.2 常用条件格式类型

**1. 突出显示单元格规则**
- 大于/小于/等于指定值
- 介于两个值之间
- 包含特定文本
- 重复值

**2. 数据条**
- 在单元格内显示横向条形图
- 条形长度与数值大小成正比

**3. 色阶**
- 根据数值范围显示不同颜色
- 常用于热图效果

**4. 图标集**
- 根据数值显示不同图标（箭头、星星等）

### 3.3 示例：标记销售业绩

**需求：** 将销售额大于10000的单元格标记为绿色背景，小于5000的标记为红色背景。

**步骤：**
1. 选中销售额列（如 E2:E51）
2. 「条件格式」→「突出显示单元格规则」→「大于」
3. 输入 10000，选择「绿色填充」
4. 再次点击「条件格式」→「突出显示单元格规则」→「小于」
5. 输入 5000，选择「红色填充」

## 四、样式与模板

### 4.1 单元格样式

**使用预设样式：**
1. 选中单元格
2. 「开始」选项卡 → 「单元格样式」
3. 选择合适的样式

**自定义样式：**
1. 「单元格样式」→「新建单元格样式」
2. 输入样式名称
3. 点击「格式」设置字体、边框、填充等
4. 点击「确定」

### 4.2 模板

**创建模板：**
1. 打开一个已格式化好的工作簿
2. 「文件」→「另存为」
3. 选择保存类型为「Excel 模板 (*.xltx)」
4. 输入模板名称并保存

**使用模板：**
1. 「文件」→「新建」
2. 点击「个人」或浏览到模板位置
3. 选择模板创建新工作簿

## 五、常见错误与技巧

### 5.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 身份证号显示为科学计数法 | 默认数字格式 | 设置为文本格式 |
| 日期显示为数字 | 数字格式问题 | 设置为日期格式 |
| 条件格式不生效 | 条件设置错误 | 检查条件规则 |

### 5.2 实用技巧

1. **批量设置格式**：选中多个单元格，一次性设置格式
2. **格式刷**：复制一个单元格的格式到其他单元格
3. **清除格式**：选中单元格 → 「清除」→「清除格式」
4. **快速格式化数字**：`Ctrl+Shift+1`（千位分隔符）、`Ctrl+Shift+4`（货币）、`Ctrl+Shift+5`（百分比）
