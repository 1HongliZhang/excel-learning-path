# 基础公式与函数

## 一、公式基础

### 1.1 公式的基本概念

公式是 Excel 中用于计算的表达式，以等号 `=` 开头。

**示例：**
```
=1+2         // 结果：3
=A1+B1       // 将A1和B1单元格的值相加
=SUM(A1:A10) // 求和A1到A10
```

### 1.2 相对引用与绝对引用

**相对引用（默认）：**
- 公式复制到其他单元格时，引用会自动调整
- 示例：`=A1+B1` 复制到 C2 变为 `=A2+B2`

**绝对引用：**
- 使用 `$` 符号锁定行或列
- `$A$1`：锁定列A和行1
- `$A1`：锁定列A，行可变
- `A$1`：锁定行1，列可变

**示例场景：**
```
=A1*$B$1   // B1始终不变，A1会随位置变化
=SUM($A$1:$A$10)  // 始终引用A1到A10
```

### 1.3 混合引用

结合相对引用和绝对引用：
- `$A1`：列固定，行变化
- `A$1`：行固定，列变化

## 二、常用基础函数

### 2.1 SUM - 求和

**语法：**
```
SUM(number1, [number2], ...)
```

**示例：**
```
=SUM(A1:A10)           // 求和A1到A10
=SUM(A1, B1, C1)       // 求和A1、B1、C1
=SUM(A1:A10, B1:B10)   // 求和两个区域
```

### 2.2 AVERAGE - 平均值

**语法：**
```
AVERAGE(number1, [number2], ...)
```

**示例：**
```
=AVERAGE(A1:A10)       // 计算A1到A10的平均值
=AVERAGE(1, 2, 3, 4)   // 计算1、2、3、4的平均值
```

### 2.3 COUNT - 计数

**语法：**
```
COUNT(value1, [value2], ...)
```

**注意：** COUNT 只计算包含数字的单元格。

**示例：**
```
=COUNT(A1:A10)         // 统计A1到A10中数字的个数
=COUNT(A1:A10, B1:B5)  // 统计两个区域中数字的个数
```

### 2.4 COUNTA - 非空单元格计数

**语法：**
```
COUNTA(value1, [value2], ...)
```

**示例：**
```
=COUNTA(A1:A10)        // 统计A1到A10中非空单元格个数
```

### 2.5 MIN - 最小值

**语法：**
```
MIN(number1, [number2], ...)
```

**示例：**
```
=MIN(A1:A10)           // 找出A1到A10中的最小值
```

### 2.6 MAX - 最大值

**语法：**
```
MAX(number1, [number2], ...)
```

**示例：**
```
=MAX(A1:A10)           // 找出A1到A10中的最大值
```

### 2.7 IF - 条件判断

**语法：**
```
IF(logical_test, [value_if_true], [value_if_false])
```

**示例：**
```
=IF(A1>100, "合格", "不合格")  // 如果A1大于100显示"合格"，否则显示"不合格"
=IF(B2>=60, "及格", IF(B2>=40, "补考", "不及格"))  // 嵌套IF
```

## 三、公式审核

### 3.1 公式错误类型

| 错误代码 | 含义 | 常见原因 |
|----------|------|----------|
| `#VALUE!` | 值错误 | 数据类型不匹配 |
| `#REF!` | 引用错误 | 引用的单元格被删除 |
| `#DIV/0!` | 除零错误 | 除数为0 |
| `#N/A` | 不可用 | VLOOKUP等函数找不到匹配值 |
| `#NAME?` | 名称错误 | 函数名拼写错误 |
| `#NUM!` | 数字错误 | 数值超出范围 |

### 3.2 审核工具

**1. 显示公式：**
- 「公式」选项卡 → 「显示公式」
- 或按 `Ctrl+`（反引号）

**2. 追踪引用单元格：**
- 选中包含公式的单元格
- 「公式」选项卡 → 「追踪引用单元格」
- 蓝色箭头指向公式引用的单元格

**3. 追踪从属单元格：**
- 选中单元格
- 「公式」选项卡 → 「追踪从属单元格」
- 蓝色箭头指向引用该单元格的公式

**4. 错误检查：**
- 「公式」选项卡 → 「错误检查」
- Excel 会检查并提示公式中的错误

### 3.3 公式求值

**操作步骤：**
1. 选中包含公式的单元格
2. 「公式」选项卡 → 「公式求值」
3. 点击「求值」逐步查看计算过程

## 四、示例练习

### 4.1 销售额计算

**场景：** 计算每个产品的销售额（销售量 × 单价）

**数据：**
| 产品 | 销售量 | 单价 | 销售额 |
|------|--------|------|--------|
| 产品A | 100 | 50 | ? |
| 产品B | 200 | 30 | ? |

**公式：**
```
=B2*C2   // 在D2单元格输入
```

### 4.2 业绩评级

**场景：** 根据销售额进行评级：
- >= 10000：优秀
- 5000-9999：良好
- < 5000：一般

**公式：**
```
=IF(D2>=10000, "优秀", IF(D2>=5000, "良好", "一般"))
```

## 五、常见错误与技巧

### 5.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 公式显示为文本 | 单元格格式为文本 | 设置为常规格式，重新输入公式 |
| 复制公式后结果错误 | 引用未正确锁定 | 使用绝对引用 `$` |
| 求和结果为0 | 数据为文本格式 | 转换为数字格式 |

### 5.2 实用技巧

1. **快速求和**：选中数据区域，Excel 状态栏会显示求和、平均值等
2. **自动求和**：点击「自动求和」按钮，Excel 自动选择相邻区域
3. **公式提示**：输入函数名时，Excel 会显示参数提示
4. **函数参数工具**：按 `Ctrl+Shift+A` 显示函数参数对话框
