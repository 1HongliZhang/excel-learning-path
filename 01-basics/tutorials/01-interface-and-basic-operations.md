# Excel 界面与基本操作

## 一、Excel 界面概览

### 1.1 工作簿（Workbook）

工作簿是 Excel 的文件单位，扩展名为 `.xlsx`（普通文件）或 `.xlsm`（包含宏的文件）。一个工作簿可以包含多个工作表。

### 1.2 工作表（Worksheet）

工作表是工作簿中的页面，默认有 Sheet1、Sheet2、Sheet3。

**工作表操作：**
- **新建工作表**：点击底部「+」按钮
- **重命名工作表**：双击工作表标签，输入新名称
- **删除工作表**：右键点击标签 → 删除
- **移动/复制工作表**：右键点击标签 → 移动或复制

### 1.3 单元格（Cell）

单元格是工作表的最小单位，由列字母和行数字标识（如 A1、B3）。

**单元格选择：**
- 点击单个单元格
- 拖拽选择连续区域（如 A1:C10）
- `Ctrl+A` 全选
- `Shift+方向键` 扩展选择

## 二、行列操作

### 2.1 插入与删除

**插入行/列：**
1. 选中要插入位置的行号或列标
2. 右键 → 插入

**删除行/列：**
1. 选中要删除的行号或列标
2. 右键 → 删除

### 2.2 调整行高与列宽

**方法一：**
1. 鼠标移到行号/列标之间的分隔线
2. 双击自动调整到合适大小
3. 拖拽手动调整

**方法二：**
1. 选中行/列
2. 右键 → 行高/列宽
3. 输入具体数值

## 三、冻结窗格

当数据量较大时，冻结窗格可以在滚动时保持标题行或列可见。

**操作步骤：**
1. 选中要冻结位置的下一行（或右一列）
2. 「视图」选项卡 → 「冻结窗格」
3. 选择「冻结首行」、「冻结首列」或「冻结拆分窗格」

**示例：**
- 要冻结第1-3行：选中第4行 → 冻结拆分窗格
- 要冻结A列：选中B列 → 冻结拆分窗格

## 四、基本快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+X` | 剪切 |
| `Ctrl+S` | 保存 |
| `Ctrl+A` | 全选 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Enter` | 确认输入，移动到下一行 |
| `Tab` | 移动到下一列 |
| `Shift+Enter` | 确认输入，移动到上一行 |
| `Ctrl+方向键` | 快速移动到数据边界 |

## 五、保存与分享

### 5.1 保存文件

**保存为新文件：**
1. 「文件」选项卡 → 「保存」或「另存为」
2. 选择保存位置
3. 输入文件名
4. 选择文件格式

**常用文件格式：**
- `.xlsx`：Excel 工作簿（推荐）
- `.xlsm`：包含宏的工作簿
- `.xls`：Excel 97-2003 格式（兼容旧版本）

### 5.2 分享工作簿

**方法一：通过 OneDrive**
1. 「文件」选项卡 → 「共享」
2. 点击「上传到 OneDrive」
3. 设置共享权限

**方法二：发送附件**
1. 「文件」选项卡 → 「共享」
2. 点击「附加到电子邮件」

## 六、常见错误与技巧

### 6.1 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 文件无法打开 | 文件损坏或版本不兼容 | 使用「打开并修复」功能 |
| 工作表标签消失 | 被隐藏 | 右键任意标签 → 取消隐藏 |
| 单元格内容显示 `####` | 列宽不够 | 调整列宽 |

### 6.2 实用技巧

1. **快速切换工作表**：`Ctrl+PageUp/PageDown`
2. **新建空白工作簿**：`Ctrl+N`
3. **快速访问最近文件**：「文件」→「最近」
4. **设置默认保存位置**：「文件」→「选项」→「保存」
