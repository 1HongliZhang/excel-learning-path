# 基础阶段 - Excel 入门与核心操作

## 阶段目标

本阶段旨在帮助初学者快速掌握 Excel 的基本操作和核心功能，为后续进阶学习打下坚实基础。

## 学习内容

1. **Excel 界面与基本操作** - 认识工作簿、工作表、单元格，掌握行列操作、冻结窗格等技巧
2. **数据录入与格式化** - 学习快速填充、数据格式设置、条件格式等
3. **基础公式与函数** - 掌握相对/绝对引用、SUM、AVERAGE、COUNT、IF等常用函数

## 适用版本

- Excel 2016+
- Microsoft 365（推荐）
