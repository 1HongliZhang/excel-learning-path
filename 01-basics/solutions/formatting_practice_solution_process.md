# formatting_practice.xlsx 练习解决方案

## 练习目标

1. 设置数据格式（日期、货币）
2. 添加条件格式标记薪资高低
3. 使用IF函数判断薪资等级

## 数据说明

- **列A（姓名）**：员工姓名
- **列B（入职日期）**：入职日期
- **列C（部门）**：所属部门
- **列D（薪资）**：月薪

## 解决方案步骤

### Step 1: 设置日期格式

**目标**：将入职日期列设置为标准日期格式

**操作**：
1. 选中B2:B51区域
2. 右键 → 「设置单元格格式」（或按 `Ctrl+1`）
3. 在「数字」选项卡中选择「日期」
4. 选择格式类型：`2024/01/15`
5. 点击「确定」

### Step 2: 设置薪资格式

**目标**：将薪资列设置为货币格式

**操作**：
1. 选中D2:D51区域
2. 右键 → 「设置单元格格式」（或按 `Ctrl+1`）
3. 在「数字」选项卡中选择「货币」
4. 设置小数位数为0，货币符号为¥，千位分隔符打勾
5. 点击「确定」

### Step 3: 添加条件格式标记薪资

**目标**：用不同颜色标记薪资高低

**操作**：

**规则1：薪资≥15000标记绿色背景**
1. 选中D2:D51区域
2. 「开始」选项卡 → 「条件格式」→「突出显示单元格规则」→「大于或等于」
3. 在弹出的对话框中输入 `15000`
4. 选择「自定义格式」
5. 点击「填充」选项卡，选择绿色
6. 点击「确定」两次

**规则2：薪资<8000标记红色背景**
1. 保持D2:D51区域选中
2. 「开始」选项卡 → 「条件格式」→「突出显示单元格规则」→「小于」
3. 在弹出的对话框中输入 `8000`
4. 选择「自定义格式」
5. 点击「填充」选项卡，选择红色
6. 点击「确定」两次

### Step 4: 使用色阶显示薪资分布

**目标**：用渐变颜色显示薪资分布情况

**操作**：
1. 选中D2:D51区域
2. 「开始」选项卡 → 「条件格式」→「色阶」
3. 选择一个合适的色阶样式（如绿色-黄色-红色）
4. Excel会自动根据薪资值显示不同深浅的颜色

### Step 5: 添加薪资等级列

**目标**：用IF函数判断薪资等级

**操作**：
1. 在E1单元格输入「薪资等级」
2. 选中E2单元格
3. 输入公式：`=IF(D2>=15000,"高薪",IF(D2>=8000,"中薪","底薪"))`
4. 按Enter键确认
5. 鼠标移到E2单元格右下角，当指针变为「+」时，双击或拖拽填充到E51

**公式说明**：
- `IF(D2>=15000,"高薪",...)`：如果薪资≥15000，显示"高薪"
- `IF(D2>=8000,"中薪","底薪")`：否则如果薪资≥8000，显示"中薪"，否则显示"底薪"

### Step 6: 冻结首行

**目标**：滚动时保持标题行可见

**操作**：
1. 选中第2行
2. 「视图」选项卡 → 「冻结窗格」→「冻结拆分窗格」

### Step 7: 调整列宽

**目标**：使所有内容完整显示

**操作**：
1. 选中A列到E列
2. 鼠标移到任意两列之间的分隔线
3. 双击自动调整列宽

## 最终效果

完成后的数据表应包含：
- B列显示标准日期格式（如2024/01/15）
- D列显示货币格式（如¥12,000）
- D列根据薪资显示不同颜色（绿色-高薪，红色-底薪）
- E列显示薪资等级（高薪/中薪/底薪）
- 标题行固定在顶部
- 所有内容完整显示

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 日期显示为数字 | 单元格格式为常规 | 设置为日期格式 |
| 薪资格式不显示千位分隔符 | 格式设置错误 | 检查货币格式设置 |
| IF函数返回错误 | 括号不匹配 | 检查公式中的括号数量 |
