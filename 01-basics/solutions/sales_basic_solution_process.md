# sales_basic.xlsx 练习解决方案

## 练习目标

1. 计算销售额（销售量 × 单价）
2. 处理空白单元格
3. 添加条件格式标记销售额高低

## 数据说明

- **列A（日期）**：销售日期
- **列B（产品）**：产品名称
- **列C（销售量）**：销售数量，部分单元格为空
- **列D（单价）**：产品单价，部分单元格为空
- **列E（销售额）**：空白，需要用公式计算
- **列F（区域）**：销售区域

## 解决方案步骤

### Step 1: 计算销售额

**目标**：在E列计算每个产品的销售额

**操作**：
1. 选中E2单元格
2. 输入公式：`=IF(AND(C2<>"", D2<>""), C2*D2, "")`
3. 按Enter键确认
4. 鼠标移到E2单元格右下角，当指针变为「+」时，双击或拖拽填充到E51

**公式说明**：
- `IF(AND(C2<>"", D2<>""), C2*D2, "")`：如果销售量和单价都不为空，则计算乘积，否则显示空
- 也可以直接用 `=C2*D2`，当C2或D2为空时会显示0

### Step 2: 设置销售额格式

**目标**：将销售额列设置为货币格式

**操作**：
1. 选中E2:E51区域
2. 右键 → 「设置单元格格式」（或按 `Ctrl+1`）
3. 在「数字」选项卡中选择「货币」
4. 设置小数位数为2，货币符号为¥
5. 点击「确定」

### Step 3: 添加条件格式

**目标**：标记销售额高低

**操作**：

**规则1：销售额大于10000标记绿色背景**
1. 选中E2:E51区域
2. 「开始」选项卡 → 「条件格式」→「突出显示单元格规则」→「大于」
3. 在弹出的对话框中输入 `10000`
4. 选择「绿色填充」
5. 点击「确定」

**规则2：销售额小于5000标记红色背景**
1. 保持E2:E51区域选中
2. 「开始」选项卡 → 「条件格式」→「突出显示单元格规则」→「小于」
3. 在弹出的对话框中输入 `5000`
4. 选择「红色填充」
5. 点击「确定」

### Step 4: 冻结首行

**目标**：滚动时保持标题行可见

**操作**：
1. 选中第2行
2. 「视图」选项卡 → 「冻结窗格」→「冻结拆分窗格」

### Step 5: 调整列宽

**目标**：使所有内容完整显示

**操作**：
1. 选中A列到F列
2. 鼠标移到任意两列之间的分隔线
3. 双击自动调整列宽

## 最终效果

完成后的数据表应包含：
- E列显示正确计算的销售额（货币格式）
- 销售额>10000的单元格显示绿色背景
- 销售额<5000的单元格显示红色背景
- 标题行固定在顶部
- 所有内容完整显示

## 常见问题与解决方案

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 销售额显示为0 | 销售量或单价为空 | 使用IF函数判断非空 |
| 条件格式不生效 | 条件设置错误 | 检查条件值是否正确 |
| 列宽不够显示#### | 列宽不足 | 调整列宽 |
